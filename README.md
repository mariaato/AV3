# Av2 - Avaliação Prática PHP: Sistema de Cadastro de Clientes, Login e Carrinho de Compras

Contextualização:
O objetivo desta avaliação é desenvolver um sistema de cadastro de clientes, login, gerenciamento de
arquivos, e carrinho de compras. O sistema envolve o gerenciamento de clientes, incluindo o upload de foto
de perfil e documento PDF obrigatório, além de um carrinho de compras persistido utilizando cookies. Os
clientes podem realizar compras e gerenciar seus dados e perfis.
